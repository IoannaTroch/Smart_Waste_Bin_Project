## Project Overview
This project aims to implement an intelligent waste management system. Using a **Raspberry Pi** and **Motion Sensors**, the system detects waste disposal activities and transmits data to a **Middleware layer** for processing and visualization.

## Key Features
- **Edge Computing:** Data acquisition using PIR/Motion sensors on Raspberry Pi.
- **Middleware Integration:** Reliable data collection and messaging.
- **Data Processing:** Context-aware modeling and virtual sensors.
- **Scalability:** Fully containerized deployment using Docker.
- **Visualization:** Real-time dashboard for monitoring bin status.
